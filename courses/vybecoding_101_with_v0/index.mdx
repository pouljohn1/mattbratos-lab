## Who this course is for

This course is designed for total beginners who have never built a webpage before and want to ship something in under 20 minutes. 

You do **not** need to have any prior coding experince.

## What you will learn

by the end of this course you will be able to: 
* explain what v0 and vercel is 
* explain basics of hosting and deployment
* generate and iterate on a landing page using v0 prompts
* deploy your site to Vercel and verify it works
* post your project on HackLab to show you work to other hackers

## Course structure

This course is 1 module you should complete in around `20 min`

### Module 1: Build and ship a landing page

1. Theory crash course: v0 + Vercel basics (`~5 min`)
2. Build: generate + edit in v0(`~10 min`)
3. Ship: deploy to Vercel + post on HackLab (`~5min`)
